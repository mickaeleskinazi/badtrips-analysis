# Cadre phénoménologique

## Objectif

La partie phénoménologique ne doit pas seulement lister des symptômes. Elle doit décrire la dynamique vécue du bad trip : comment une altération corporelle, perceptive ou cognitive devient une crise de sens, puis parfois une expérience intégrée.

## Ancrage dans les études existantes

### Challenging Experience Questionnaire

Le Challenging Experience Questionnaire (CEQ) propose sept dimensions utiles pour structurer les expériences difficiles avec la psilocybine : peur, grief, mort, folie, isolement, détresse physique et paranoïa.

Usage dans notre corpus :

- `fear_of_death` correspond à la dimension death ;
- `fear_of_madness` correspond à insanity ;
- `interoceptive_threat` et plusieurs AE-like markers correspondent à physical distress ;
- `paranoia_social_threat` correspond à paranoia ;
- `social_support` et `context_alone` permettent d'approcher l'isolement.

Le CEQ peut servir de grille de comparaison, mais notre corpus permet d'ajouter une dimension narrative absente des questionnaires : la séquence crise -> résistance -> bascule -> résolution.

### Ego dissolution et rupture du soi

L'Ego-Dissolution Inventory (EDI) permet d'opérationnaliser la dissolution des frontières du soi. Pour notre analyse, l'enjeu n'est pas seulement la présence d'ego dissolution, mais sa valence :

- dissolution vécue comme libération ;
- dissolution vécue comme menace ;
- dissolution convertie en peur de mourir ;
- dissolution convertie en acceptation/surrender.

Cette distinction est centrale pour éviter de confondre intensité psychédélique et bad trip.

### Mystical experience et intégration

Les travaux sur le Mystical Experience Questionnaire (MEQ) montrent que certains aspects de l'expérience psychédélique, notamment unité, sacré, noétique, transcendance du temps/espace, ineffabilité et humeur positive, peuvent prédire sens personnel et changements positifs.

Usage dans notre corpus :

- `integration_positive_afterwards` capte une version narrative très approximative de cette intégration ;
- `acceptance_surrender` peut être traité comme une passerelle entre crise et intégration ;
- les reports avec peur de mourir + let go + intégration doivent être lus qualitativement.

### Near-death-like experiences

Les études sur DMT et expériences de mort imminente soutiennent l'idée qu'une expérience de mort subjective peut être psychologiquement structurante. Notre hypothèse est plus précise :

> La peur de mourir devient potentiellement transformatrice seulement si elle est reconfigurée par l'acceptation, le soutien ou une relecture intégrative.

### REBUS, predictive processing et interoception

Le modèle REBUS propose que les psychédéliques relâchent la précision des croyances de haut niveau, ce qui augmente l'influence de signaux bottom-up. Couplé à l'interoception, cela permet de formuler le bad trip comme crise de prédiction :

```text
signaux corporels/perceptifs inhabituels
-> difficulté à les expliquer
-> interprétation catastrophique
-> perte de contrôle
-> peur de mourir / devenir fou / ne pas revenir
-> possible bascule par let go
```

## Structure proposée pour la section phénoménologique

### 1. La crise d'interoception menaçante

Décrire comment le corps devient illisible ou menaçant : coeur, respiration, nausée, tremblements, sensation d'overdose, peur que le corps lâche.

Lien avec les résultats :

- `interoceptive_threat` prédit fortement `fear_of_death` ;
- cannabis naturel et cannabinoïdes synthétiques ressortent fortement sur cet axe.

### 2. La peur de mourir comme point de bascule

La peur de mourir peut venir :

- du corps ;
- de la dissolution du soi ;
- de la déréalisation ;
- de l'impression que l'état ne finira jamais.

Il faut distinguer :

- peur de mourir somatique ;
- mort psychique/ego death ;
- mort métaphysique ou cosmique ;
- peur d'une mort imminente nécessitant aide médicale.

### 3. Perte de contrôle et résistance

La résistance correspond aux efforts pour rétablir l'ancien modèle du monde :

- contrôler les pensées ;
- vérifier le corps ;
- appeler à l'aide ;
- fuir le setting ;
- se convaincre que l'expérience va finir.

Cette phase est importante narrativement car elle précède souvent l'escalade.

### 4. Let go / surrender

Le `let go` doit être codé comme événement de bascule :

- abandon du contrôle ;
- acceptation de mourir ;
- acceptation de ne pas comprendre ;
- consentement à traverser l'expérience ;
- transformation de la menace en processus.

Il faut distinguer le let go de :

- résignation ;
- épuisement ;
- black-out ;
- dissociation ;
- sédation ou intervention médicale.

### 5. Résolution et intégration

Après la crise :

- résolution sans sens particulier ;
- honte ou traumatisation ;
- prudence/apprentissage ;
- insight personnel ;
- transformation thérapeutique potentielle.

## Méthode qualitative proposée

1. Sélectionner un échantillon stratifié :
   - peur de mourir + let go + intégration ;
   - peur de mourir sans let go ;
   - interoception forte sans intégration ;
   - cannabinoïdes synthétiques ;
   - psilocybine/LSD/DMT/Salvia.
2. Lire les reports complets.
3. Annoter la séquence narrative :
   - onset ;
   - montée ;
   - moment de menace ;
   - pic de peur ;
   - résistance ;
   - bascule ;
   - résolution ;
   - aftermath.
4. Comparer les séquences par substance.
5. Réviser les marqueurs lexicaux à partir des faux positifs/faux négatifs.

## Hypothèse phénoménologique finale

Les bad trips ne sont pas seulement des réactions négatives à une substance. Ce sont des crises de stabilisation du corps, du soi et du monde. La peur de mourir est le point de condensation de cette crise. Le potentiel thérapeutique apparaît lorsque la menace est reconfigurée par un processus de let go, de soutien ou d'intégration.
