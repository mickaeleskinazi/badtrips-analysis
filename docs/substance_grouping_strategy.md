# Stratégie d'agrégation des substances

## Principe

Les catégories Erowid contiennent beaucoup de redondances : familles générales, molécules, espèces de champignons, produits commerciaux, catégories éditoriales et combinaisons. Pour l'analyse, il faut donc travailler avec plusieurs niveaux d'agrégation.

## Niveaux recommandés

### Niveau 1 : grandes familles

Utile pour une première carte du corpus :

- psychédéliques classiques ;
- psychédéliques phenethylamines ;
- entactogènes ;
- cannabis / cannabinoïdes ;
- stimulants ;
- dissociatifs ;
- délirogènes ;
- dépresseurs / sédatifs ;
- opioïdes ;
- alcohol ;
- pharmaceuticals.

### Niveau 2 : groupes ciblés

Utile pour l'article principal, car il limite les redondances tout en gardant des profils interprétables :

- `psilocybin_mushrooms` : tous les champignons psilocybine, indépendamment de l'espèce ;
- `lsd_lysergamides` : LSD, 1P-LSD, AL-LAD, ALD-52, autres lysergamides ;
- `dmt_ayahuasca_5meo` : DMT, 5-MeO-DMT, ayahuasca, pharmahuasca, plantes DMT ;
- `other_tryptamines` : 4-AcO-DMT, 4-HO-MET, DPT, DiPT, MiPT, etc. ;
- `mescaline_cacti` : mescaline, peyote, San Pedro, cacti ;
- `phenethylamines_2c_nbome_dox` : 2C-x, NBOMe, DOx ;
- `mdma_entactogens` : MDMA, MDA, MDEA, methylone, 6-APB ;
- `amphetamine_like_stimulants` : amphétamine, méthamphétamine, cathinones, methylphenidate, MDPV ;
- `cocaine_crack` ;
- `cannabis_natural` : cannabis, hash, high THC, high CBD ;
- `synthetic_cannabinoids` : cannabinoid receptor agonists, Spice/K2/JWH/AM-2201 ;
- `ketamine_pcp_arylcyclohexylamines` ;
- `dxm` ;
- `salvia` ;
- `deliriants_anticholinergics` ;
- `benzodiazepines_zdrugs_sedatives` ;
- `opioids` ;
- `alcohol`.

## Pourquoi ne pas commencer par les symptômes seuls ?

Les symptômes ou dimensions phénoménologiques doivent être l'axe principal de l'interprétation, mais pas l'unique axe de stratification. Une analyse uniquement par symptômes répondrait à la question : "quelles formes prend le bad trip ?" Elle ne répondrait pas à : "les molécules ouvrent-elles vers des formes différentes de crise ?"

La stratégie recommandée est donc une matrice :

```text
groupes de substances x dimensions phénoménologiques
```

Les groupes de substances indiquent l'exposition approximative ; les dimensions phénoménologiques indiquent la forme vécue de la crise.

## Focalisation proposée

Pour un premier article, il serait raisonnable de se concentrer sur :

1. psilocybine/champignons ;
2. LSD/lysergamides ;
3. DMT/ayahuasca/5-MeO-DMT ;
4. autres tryptamines ;
5. phenethylamines/2C/NBOMe/DOx ;
6. MDMA/entactogènes ;
7. cannabis naturel ;
8. cannabinoïdes synthétiques ;
9. dissociatifs principaux ;
10. délirogènes.

Les opioïdes, benzodiazépines, alcohol et pharmaceuticals peuvent servir de contraste toxicologique/médical, ou être réservés à un second article.

## Limites

- Les reports multi-substances peuvent appartenir à plusieurs groupes.
- Les catégories Erowid ne garantissent pas la substance réellement active.
- Les doses, contextes et antécédents sont incomplets.
- Les regroupements doivent être révisés après lecture qualitative d'échantillons.
