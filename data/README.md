# Données

Ce dossier peut contenir localement le corpus de travail, mais les fichiers CSV bruts ne doivent pas être commités.

Fichier attendu par les scripts :

```text
data/badtrip_reports_by_substance.csv
```

Raison : les reports Erowid sont sous copyright et leurs conditions d'utilisation restreignent la réutilisation, l'analyse automatisée et la redistribution sans permission écrite. Le dépôt doit donc rester centré sur le code, les méthodes et les résultats agrégés non reconstructibles.
